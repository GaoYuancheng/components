展开关闭交互按钮。

## API

| 字段名     | 说明               | 类型                        | 默认值 |
| :--------- | :----------------- | :-------------------------- | :----: |
| classNames | 类名               | string                      |   -    |
| style      | CSS 样式           | `React.CSSProperties`       |   -    |
| open       | 必填，是否展开状态 | `boolean`                   |   -    |
| onClick    | 点击时的事件       | `(isOpen: boolean) => void` |   -    |
